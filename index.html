Blockchain

When we say the words “block” and “chain” in this context, we are actually talking about digital information (the “block”) stored in a public database (the “chain”).
“Blocks” on the blockchain are made up of pieces of information. Especially, they have three parts. 1.Information about transactions/ 2.Information about who is taking part in transactions/ 3.Information that distinguishes them from other blocks/
Hash codes are created by a math function that turns digital information into a string of numbers and letters.

How does it work?

  1. A transaction must occur.
  2. That transaction must be verified.
  3. That transaction must be stored in a block.
  4. That block must be given a hash.
In order for a block to be added to the blockchain, however, 4 things must happen.
The block is also given the hash of the most recent block added to the blockchain. Once hashed, the block can be added to the blockchain.

Where Is the Blockchain Stored?

Anyone can view the contents of the blockchain, but users can also opt to connect their computers to the blockchain network.
Their computer receives a copy of the blockchain that is updated automatically whenever a new block is added. As each copy of the blockchain is identical, spreading that information across a network of computers makes the information more difficult to manipulate.

Blockchain Security

Transactions on blockchain are not completely anonymous, personal information about users is limited to their digital signature. If you cannot know who is adding blocks to the blockchain, how can you trust blockchain?
After a block has been added to the end of the blockchain, it is very difficult to go back and change the contents of the block. That’s because each block contains its own hash, along with the hash of the block before it.
To address the issue of trust, blockchain networks have implemented tests for computers that want to join and add blocks to the chain. The tests, called “consensus models,” require users to “prove” themselves before they can participate in a blockchain network.
